Python Exam Cheat Sheet

I. Input & Data Conversion

  • Read a single line of string: 
    s = input()
  • Read an integer: 
    n = int(input())
  • Read a float: 
    f = float(input())
  • Read a list of space-separated integers: 
    nums = list(map(int, input().split()))
    Example: 1 2 3 becomes [1, 2, 3]
  • Read a list of space-separated strings: 
    words = input().split()
    Example: hello world becomes ['hello', 'world']

II. Loops & Iteration

  • for loop (by range): 
    for i in range(5): # 0, 1, 2, 3, 4
    # ...
for i in range(1, 6): # 1, 2, 3, 4, 5
    # ...
for i in range(5, 0, -1): # 5, 4, 3, 2, 1
    # ...
  • for loop (through a list/set/etc.): 
    my_collection = [10, 20, 30] # Works for lists, sets, strings, dict keys, etc.
for item in my_collection:
    # ...
  • for loop with index (using enumerate): 
    my_list = ['a', 'b', 'c']
for index, value in enumerate(my_list):
    # ... index 0, value 'a'; index 1, value 'b', etc.
  • while loop: 
    count = 0
while count < 5:
    # ...
    count += 1
  • Loop Control:
    • break: Exits the current loop.
    • continue: Skipped rest of current iter & goes to next.

III. Conditional Statements

  • if, elif, else: 
    if condition1:
    # ...
elif condition2:
    # ...
else:
    # ...
  • Logical Operators:
    • and: Both conditions must be true.
    • or: At least one condition must be true.
    • not: Reverses the boolean value.

IV. Lists (Common Operations)

  • Create: my_list = [] or my_list = [1, 2, 3]
  • Access: my_list[0] (first), my_list[-1] (last)
  • Add:
    • my_list.append(item): Adds to end.
    • my_list.insert(index, item): Inserts at specific index.
  • Remove:
    • my_list.pop(): Removes and returns last item.
    • my_list.pop(index): Removes and returns item at index.
    • my_list.remove(value): Removes first occurrence of value.
  • Length: len(my_list)
  • Sort:
    • my_list.sort(): Sorts in-place (ascending).
    • my_list.sort(reverse=True): Sorts in-place (descending).
    • sorted(my_list): Returns a new sorted list.
  • Min/Max/Sum: min(my_list), max(my_list), sum(my_list)

V. Sets (Unique Elements & Fast Lookups)

  • Create:
    • my_set = set(): Empty set.
    • my_set = {1, 2, 3}: From literal values.
    • my_set = set([1, 2, 2, 3]): From a list (duplicates removed). Result: {1, 2, 3}
  • Add:
    • my_set.add(item): Adds an item. No effect if item present.
  • Remove:
    • my_set.remove(item): Removes item. Raises KeyError if not found.
    • my_set.discard(item): Removes item. Does nothing if not found (safer).
    • my_set.pop(): Removes and returns arbitrary item (unordered).
  • Check Membership (Very Fast!): 
    if item in my_set:
    # ...
if item not in my_set:
    # ...
  • Length: len(my_set)
  • Set Operations:
    • set1.union(set2) or set1 | set2: All unique elements.
    • set1.intersection(set2) or set1 & set2: Common elements.
    • set1.difference(set2) or set1 - set2: In set1 but not set2.
    • set1.symmetric_difference(set2) or set1 ^ set2: In either, but not both.

VI. Dictionaries (Key-Value Pairs)

  • Create:
    • my_dict = {} or my_dict = dict(): Empty.
    • my_dict = {'name': 'Alice', 'age': 30}: With initial pairs.
  • Access Value:
    • value = my_dict['key']: Access by key. Raises KeyError if not found.
    • value = my_dict.get('key'): Returns None if key not found (safer).
    • value = my_dict.get('key', default_value): Returns default_value if not found.
  • Add/Update:
    • my_dict['new_key'] = 'new_value': Adds new pair.
    • my_dict['existing_key'] = 'updated_value': Updates existing.
  • Remove:
    • del my_dict['key']: Deletes pair. Raises KeyError if not found.
    • value = my_dict.pop('key'): Removes & returns value. Raises KeyError if not found.
    • value = my_dict.pop('key', default_value): Removes & returns value; default_value if not found.
  • Check Key Membership (Fast!): 
    if 'key' in my_dict:
    # ...
  • Length: len(my_dict) (number of key-value pairs)
  • Iterating:
    • Keys (default): 
      for key in my_dict:
    # ...
for key in my_dict.keys(): # Explicitly iterate keys
    # ...
    • Values: 
      for value in my_dict.values():
    # ...
    • Key-Value Pairs: 
      for key, value in my_dict.items():
    # ...

VII. Strings (Common Operations)

  • Length: len(my_string)
  • Access Character: my_string[0]
  • Slicing: my_string[start:end:step]
    • my_string[1:4] (chars from index 1 to 3)
    • my_string[2:] (from index 2 to end)
    • my_string[:5] (from start to index 4)
  • Concatenation: str1 + str2
  • Split: my_string.split(separator) (defaults to whitespace)
  • Join: 'separator'.join(list_of_strings)

VIII. Output

  • Print a single value: 
    print("Hello")
print(123)
  • Print multiple values (space-separated by default): 
    print("Name:", "Alice", "Age:", 30)
# Output: Name: Alice Age: 30
  • Print a list/tuple/set/etc. of numbers with space separators: 
    my_collection = [1, 5, 9, 2] # Works for lists, tuples, sets, keys from dicts
print(*my_collection)
# Output: 1 5 9 2 (order may vary for sets/dicts before Python 3.7)
    Alternative (for any iterable, good for non-numeric as well): 
    print(' '.join(map(str, my_collection)))
# Output: 1 5 9 2 (order may vary for sets/dicts before Python 3.7)
  • Print with f-strings (formatted string literals - Python 3.6+): 
    name = "Bob"
age = 25
print(f"Name: {name}, Age: {age}")
# Output: Name: Bob, Age: 25

IX. Useful Built-in Functions

  • len(): Length of sequence (list, string, set, dict, etc.)
  • min(), max(), sum()
  • abs(): Absolute value.
  • round(number, ndigits): Round to specified decimal places.

Tips for Exam:

  • Read the problem carefully. Understand inputs, outputs, and constraints.
  • Test with examples.
  • Start with input parsing and output formatting. This ensures basic structure.
  • Break down complex problems.
  • Indent correctly! Python relies on indentation.
