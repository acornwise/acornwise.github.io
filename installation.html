Installation Guide

Follow these steps if you see security warnings during installation.

Windows

1

Run the Installer

Double-click the .exe file.

2

Handle SmartScreen

If "Windows protected your PC" appears, click More info.

3

Run Anyway

Click the Run anyway button to finish installing.

macOS (Intel)

1

Move to Applications

Drag the app to the Applications folder. Double-click to open it.

2

Open Settings

If blocked, go to System Settings > Privacy & Security.

3

Allow the App

Scroll down to "Security". Click Open Anyway next to the Acornwise notification.

macOS (Silicon)

1

Move to Applications

Drag the app to Applications. Do not open it yet.

2

Open Terminal

Press Cmd+Space, type "Terminal", and open it.

3

Run Command

Paste this exact command:

sudo xattr -cr /Applications/AcornWise\ Learning\ Platform.app

Enter your password if prompted (it won't show while typing).
